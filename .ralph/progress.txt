# Ralph Progress Log

## Codebase Patterns

- Custom fonts go in `assets/fonts/` and are linked via `react-native.config.js`
- Theme exports are centralized in `src/theme/index.ts`
- Use Platform.select() for platform-specific font family names (iOS uses font family name, Android uses file name without extension)
- DATABASE_VERSION in database.ts must match the expected version in tests
- Tests with state transitions after fireEvent.press() may need `findByTestId` with async await
