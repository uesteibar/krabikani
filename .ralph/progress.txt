# Ralph Progress Log

## Codebase Patterns
- Services live in `src/services/` directory with an index.ts for exports
- Mocks for external libraries go in `__mocks__/<package-path>/` directory (e.g., `__mocks__/@notifee/react-native.ts`)
- Tests mirror source structure in `__tests__/` directory
- @notifee/react-native is used for local notifications on both iOS and Android
- Android notification channels should be created before displaying notifications
