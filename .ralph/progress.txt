## Codebase Patterns
- Database schema changes require migrations in `MIGRATIONS` array in `src/storage/database.ts`
- When adding new database columns, update both `DatabaseSubject` and `SubjectInput` interfaces
- For API data extraction, update `convertSubjectToInput()` in `src/sync/syncService.ts`
- Component exports go in `src/components/index.ts`, utility exports in `src/utils/index.ts`
- React Native Image.prefetch() provides caching - no need for separate file system caching
- Tests with `upsertSubject()` need all fields from `SubjectInput` interface
- For radicals without Unicode characters, WaniKani provides images via `character_images` array
