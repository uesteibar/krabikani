## Codebase Patterns
- The UpcomingReviewsChart component displays cumulative totals in format "+X (Y)" where X is new reviews in that hour and Y is cumulative total
- Dashboard data is loaded via loadDashboardData() which uses Promise.all to fetch multiple data sources
- Component props follow pattern of optional values with sensible defaults (e.g., `currentPendingCount = 0`)
- Tests use @testing-library/react-native with patterns like createTestData() helpers
- User settings use getSetting/setSetting from storage - values stored as JSON strings, support boolean/string/number types
