## Codebase Patterns
- Quiz components (LessonQuiz, ReviewSession) have nearly identical UIs but different question ordering logic
- Question types are 'meaning' or 'reading', stored in QuestionType/ReviewQuestionType
- Subject colors use COLORS.neutral.black and COLORS.neutral.white from theme/colors.ts
- Styles use StyleSheet.create with COLORS, SPACING, BORDER_RADIUS from theme
- Test files use testID attributes for querying elements
- Pre-existing test failures exist in progress text format assertions and SettingsScreen tests (unrelated to new features)
- PanResponder gesture state cannot be unit tested by directly calling handlers; use structural tests to verify handlers exist, functional tests require E2E or manual verification

  - New MnemonicText component parses `<radical>`, `<kanji>`, and `<vocabulary>` tags
